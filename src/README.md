# Start server

## 1. Install packages
`npm install`

## 2. Start server
`npm run dev`

[http://localhost:5000](http://localhost:5000)

Examples api paths:\
http://localhost:5000/api/login\
http://localhost:5000/api/users\
http://localhost:5000/api/vacancies/:_id 

# Start clint

## 1. Open client folder
`cd client`

## 2. Install packages
`npm install`

## 2. Install packages
`npm start`

[http://localhost:3000](http://localhost:3000)
